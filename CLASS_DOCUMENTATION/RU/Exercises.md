# Класс: Exercises

## 1. Общая информация
*   **Имя класса:** `Exercises`
*   **Тип:** Обычный класс (Модель данных)
*   **Назначение:** Этот класс является «чертежом» для записи о завершенном упражнении. Он объединяет в один объект тип упражнения, количество ошибок, дату выполнения и имя пользователя.
*   **Взаимодействие:** В основном используется активностью `History` для хранения и отображения записей, извлеченных из базы данных.

## 2. Переменные (поля класса)
| Имя | Тип | Цель | Где используется |
| :--- | :--- | :--- | :--- |
| `exKind` | `String` | Категория (например, «Числа»). | Отображается в списке истории. |
| `exMistake` | `String` | Количество ошибок. | Отображается в списке истории. |
| `exDate` | `String` | Дата завершения. | Отображается в списке истории. |
| `exUser` | `String` | Имя пользователя. | Используется для фильтрации на экране истории. |

## 3. Методы класса

### Имя метода: `Exercises` (Конструктор)
*   **Тип метода:** `public`
*   **Логика работы:** Присваивает переданные значения полям класса.
*   **Когда вызывается:** Когда активность `History` создает новый объект записи на основе данных из базы.

### Имена методов: `get...` / `set...`
*   **Тип метода:** `public`
*   **Логика работы:** Стандартные методы для чтения или изменения данных записи.

## 4. Жизненный цикл
*   **Неприменимо:** Это простой класс для хранения данных.

## 5. Взаимодействие с интерфейсом (UI)
*   **Напрямую нет:** Его данные используются для заполнения строк в списке `ListView`.

## 6. Взаимодействие с другими компонентами
*   **Активность History:** Основной «потребитель» этого класса.
*   **HelperDB:** Структура этого класса соответствует структуре таблицы `Exercises` в базе данных.

## 7. Общая логика класса
`Exercises` — это **«Запись в журнале»**. Сам по себе он ничего не делает; он просто «держит» информацию вместе, чтобы её можно было передавать как один пакет, а не как четыре отдельные переменные.

## 8. Упрощенное объяснение
Представьте `Exercises` как **записку на холодильнике**. После того как вы закончили тренировку, приложение пишет: «Понедельник — Числа — 2 ошибки — Элия» на бумажке. Этот класс и есть тот самый листок бумаги.
