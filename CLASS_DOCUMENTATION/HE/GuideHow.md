# מחלקה: GuideHow

## 1. מידע כללי
*   **שם המחלקה:** `GuideHow`
*   **סוג:** Activity
*   **מטרת המחלקה:** מסך זה מספק מדריך חזותי על אופן השימוש בקוד מורס או באפליקציה. הוא מטמיע סרטון YouTube באמצעות `WebView`.
*   **אינטראקציה:** יורשת מ-`BaseActivity`.

## 2. משתנים (שדות המחלקה)
| שם | סוג | מטרה | היכן בשימוש |
| :--- | :--- | :--- | :--- |
| `videoID` | `String` | המזהה הייחודי של סרטון ה-YouTube. | משמש לבניית קוד ה-HTML להטמעה. |
| `webView` | `WebView` | הרכיב שמציג תוכן אינטרנטי. | משמש לטעינת נגן הווידאו. |

## 3. מתודות (שיטות) המחלקה

### שם המתודה: `onCreate`
*   **סוג המתודה:** `protected`
*   **לוגיקה:**
    1. מגדירה את העיצוב לפי `activity_guide_how.xml`.
    2. מאתחלת משתנים.
    3. מגדירה מאזין לחיצה לכפתור ה-"Back" לסגירת המסך.
    4. שולפת את ה-`videoID` מתוך משאבי המחרוזות.
    5. מגדירה את ה-`WebView` כך שיאפשר הרצת JavaScript (נדרש עבור YouTube).
    6. יוצרת מחרוזת HTML המכילה `<iframe>` עם כתובת הסרטון.
    7. טוענת את ה-HTML לתוך ה-`WebView`.
*   **מתי נקראת:** כשהמשתמש לוחץ על "איך להשתמש" בתפריט המדריך.

## 4. מחזור חיים (Lifecycle)
*   **`onCreate()`**: טוענת את נגן הווידאו.
*   **`onPause()` / `onDestroy()`**: חשוב לעצור את ה-WebView כשעוזבים את המסך (הערה: זה חסר בקוד הנוכחי).

## 5. אינטראקציה עם ממשק המשתמש (UI)
*   **WebView (`webView`):** משמש כמיכל עבור נגן הווידאו של YouTube.
*   **Button (`bBack`):** סוגר את ה-Activity.

## 6. אינטראקציה עם רכיבים אחרים
*   **דפדפן/אינטרנט:** ה-`WebView` דורש חיבור לאינטרנט כדי לטעון את הסרטון מיוטיוב.

## 7. לוגיקה כללית של המחלקה
מחלקה `GuideHow` היא **נגן וידאו**. היא משתמשת ב"דפדפן אינטרנט" זעיר בתוך האפליקציה כדי להציג סרטון ספציפי מהרשת. היא עוטפת את הסרטון בתוך קוד HTML כדי שיתאים בדיוק למסך הטלפון.

## 8. הסבר במילים פשוטות
חשבו על `GuideHow` כעל **מסך טלוויזיה קטן**. במקום לקרוא על קוד מורס, אתם יכולים לצפות בסרטון עליו. האפליקציה פותחת חלון קטן שמתחבר ליוטיוב ומציג לכם שיעור.

---
**באגים/שיפורים:**
*   **וידאו ממשיך ברקע:** בקוד הנוכחי, אם המשתמש לוחץ על כפתור הבית, הסרטון עשוי להמשיך להתנגן ברקע. עדיף לקרוא ל-`webView.onPause()` בתוך מתודת ה-`onPause()` של ה-Activity.
*   **בדיקת חיבור לאינטרנט:** אם אין אינטרנט, ה-WebView יציג שגיאה. הוספת בדיקת חיבור תשפר את חווית המשתמש.
