# 🧩 תיעוד MorseTranslator

## 1. מידע כללי
*   **שם:** `MorseTranslator`
*   **סוג:** מחלקת Java רגילה (Helper/Logic)
*   **מטרה:** מחלקה זו מכילה את המילונים והאלגוריתמים הדרושים להמרת שפה אנושית (אנגלית/מספרים) לקוד מורס ולהיפך. זהו ה"מנוע" של האפליקציה.

## 2. משתנים
| שם | סוג | מטרה |
| :--- | :--- | :--- |
| `morseMap` | `Map<String, String>` | מילון שבו ה**מפתח** הוא אות (למשל "A") וה**ערך** הוא מורס (".-"). |
| `textMap` | `Map<String, String>` | המילון ההפוך (מפתח = מורס, ערך = אות). משמש לפענוח. |

## 3. שיטות (Methods)
### שיטה: `toMorse`
*   **מה היא עושה:** 
    1. הופכת את הטקסט לאותיות גדולות (Uppercase).
    2. עוברת בלולאה על כל תו.
    3. מחפשת את התו ב-`morseMap`.
    4. מחזירה את התוצאה הסופית כמחרוזת של נקודות וקווים.

## 8. הסבר במילים פשוטות
חשבו על `MorseTranslator` כעל **מילון דו-לשוני**. זה כמו ספר שבו חצי אחד מתרגם מאנגלית למורס, והחצי השני מתרגם ממורס בחזרה לאנגלית. כשהאפליקציה צריכה לדעת איך המילה "HELLO" נשמעת, היא פותחת את ה"ספר" הזה.
