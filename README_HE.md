# EliMorse - אפליקציה ללימוד ותרגום קוד מורס

## סקירה כללית

EliMorse היא אפליקציית אנדרואיד מקיפה שתוכננה ללימוד, תרגול ויישום מעשי של קוד מורס. האפליקציה מספקת מגוון כלים למתחילים ולמשתמשים מנוסים כאחד, ומשלבת תרגילים חינוכיים עם כלי תרגום פרקטיים.

## תכונות עיקריות

- **תרגום:**
  - **טקסט לקוד מורס:** מתרגם טקסט באנגלית רגיל לייצוג בקוד מורס.
  - **קוד מורס לטקסט:** מתרגם קוד מורס חזרה לטקסט באנגלית.

- **שידור אותות:**
  - **השמעת אודיו:** מנגן כל רצף קוד מורס כסדרה של נקולות נקודות וקווים, תוך שמירה על כללי תזמון סטנדרטיים.
  - **שידור באמצעות פנס (Torch):** משדר קוד מורס בצורה ויזואלית באמצעות פנס המכשיר, עם תזמון נכון לפולסי אור והפסקות.

- **אימון ותרגילים:**
  - **תרגול קידוד:** המשתמש יכול לתרגל תרגום אותיות, מספרים ומילים לקוד מורס.
  - **תרגול האזנה:** מצב ייעודי “האזנה” שבו האפליקציה משמיעה אות מורס, והמשתמש נדרש להקליד את הטקסט המתאים.
  - **מעקב אחר טעויות:** האפליקציה עוקבת אחר טעויות במהלך התרגילים כדי לעזור למשתמש לזהות נקודות לשיפור.

- **ניהול משתמשים:**
  - **התחברות והרשמה:** מערכת אימות משתמש פשוטה לניהול נתוני המשתמש.
  - **היסטוריית תרגילים:** שומר את תוצאות התרגילים שהושלמו בפרופיל המשתמש.

- **ממשק ומשתמש:**
  - **ערכות נושא דינמיות:** תמיכה בערכות נושא בהיר, כהה והגדרת מערכת ברירת מחדל.
  - **משוב הפטי:** מספק משוב מגע בעת לחיצה על כפתורים לחוויית שימוש רספונסיבית יותר.
  - **ממשק חכם:** כפתורים לפעולות התלויות ברשת (שיחה, SMS, דוא"ל) מושבתים אוטומטית אם השירות אינו זמין.
  - **חסכון במשאבים:** רכיבים מרכזיים כמו `TextToSpeech` ו-`BroadcastReceiver` מיושמים לפי שיטות עבודה מומלצות כדי למנוע דליפות זיכרון ולהבטיח ניהול משאבים יעיל.

- **מדריכים ומידע:**
  - **טבלת קוד מורס:** רשימה בתוך האפליקציה של כל האותיות והמספרים עם המקבילות שלהם בקוד מורס.
  - **אודות:** מספק פרטי יצירת קשר ודרכים ליצור קשר עם המפתח.

## ארכיטקטורה וטכנולוגיה

- **שפת תכנות:** Java
- **פלטפורמה:** Android
- **ממשק משתמש:** פריסות XML של Android עם רכיבי Material Design.
- **מסד נתונים:** אחסון נתונים מקומי באמצעות `SQLite`.
- **רכיבים מרכזיים:** Activities, Services, BroadcastReceivers ורכיבי ממשק מותאמים אישית.
- **מודולריות:** פונקציונליות מרכזיות כמו נגן קוד מורס (`MorsePlayer`), יצירת שאלות (`QuestionRepository`) ופעולות מסד נתונים (`HelperDB`) מקובצות במחלקות עזר נפרדות לשיפור הארגון ושימוש חוזר טובים יותר.
