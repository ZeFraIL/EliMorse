# 📱 Документация ExerciseGame (Активность упражнений)

## 1. Общая информация
*   **Название:** `ExerciseGame`
*   **Тип:** Активность (Наследует `BaseActivity`)
*   **Назначение:** Это основной экран для тренировок. Он показывает пользователю задание (букву, слово или число) и просит ввести правильный код Морзе с помощью кнопок «точка», «тире» и «пробел».
*   **Взаимодействие:** Использует `MorseTranslator` для проверки ответов и `HelperDB` для сохранения итогового счета.

## 2. Переменные (Поля класса)
| Название | Тип | Назначение | Использование |
| :--- | :--- | :--- | :--- |
| `questionNum` | `int` | Номер текущего вопроса. | Увеличивается после каждого правильного ответа. |
| `failedAttempts` | `int` | Общее количество ошибок за сессию. | Сохраняется в базу данных в конце игры. |
| `stAnswer` | `String` | Строка, которую вводит пользователь. | Обновляется при нажатии на кнопки точки или тире. |
| `questions` | `List` | Список из 10 случайных вопросов. | Генерируется при запуске игры. |

## 3. Методы
### Метод: `checkAnswer`
*   **Тип:** `private`
*   **Что делает:** Сравнивает строку `stAnswer` с правильным кодом Морзе.
    *   Если **Верно**: Переходит к следующему вопросу. Если вопрос был последним — сохраняет результат и показывает диалог победы.
    *   Если **Неверно**: Увеличивает счетчик ошибок. После 3-х ошибок в одном вопросе игра заканчивается.
*   **Вызов:** При нажатии на кнопку «Ready» (Проверить).

### Метод: `saveResults`
*   **Что делает:** В фоновом потоке записывает тип игры, количество ошибок и дату в таблицу `Exercises` базы данных.

## 7. Общая логика
1. Приложение выбирает 10 случайных слов.
2. Пользователь видит слово (например, "CAT").
3. Пользователь "настукивает" код Морзе кнопками.
4. Если ответ верный, появляется следующее слово.
5. В конце игры общее количество ошибок сохраняется, чтобы пользователь мог увидеть их в разделе «История».

## 8. Объяснение простыми словами
Представьте, что `ExerciseGame` — это **цифровая викторина**. Это как учитель, который показывает вам карточки со словами. Если вы "пишете" правильный код Морзе, вы переходите к следующей карточке. В конце урока учитель записывает вашу оценку в **классный журнал** (базу данных).
