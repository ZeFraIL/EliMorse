# 🧩 Документация MorseTranslator

## 1. Общая информация
*   **Название:** `MorseTranslator`
*   **Тип:** Обычный Java-класс (Логика/Помощник)
*   **Назначение:** Этот класс содержит словари и алгоритмы для преобразования человеческого языка (букв и цифр) в азбуку Морзе и обратно. Это "движок" приложения.
*   **Взаимодействие:** Используется классами `Translate`, `ExerciseGame` и `QuestionRepository`.

## 2. Переменные (Поля класса)
| Название | Тип | Назначение |
| :--- | :--- | :--- |
| `morseMap` | `Map<String, String>` | Словарь, где **Ключ** — это буква (например, "A"), а **Значение** — код Морзе (".-"). |
| `textMap` | `Map<String, String>` | Обратный словарь (Ключ — Морзе, Значение — буква). Нужен для расшифровки. |

## 3. Методы
### Метод: `toMorse`
*   **Тип:** `public`
*   **Возвращает:** `String` (Код Морзе)
*   **Что делает:** 
    1. Переводит текст в верхний регистр.
    2. Проходит циклом по каждому символу.
    3. Ищет символ в `morseMap`.
    4. Если находит, добавляет код в результат.
    5. Возвращает готовую строку из точек и тире.

### Метод: `toText`
*   **Что делает:** Разбивает строку Морзе по пробелам. Для каждой комбинации ищет букву в `textMap`. Если комбинация неизвестна (например, "......."), добавляет символ `?`.

## 8. Объяснение простыми словами
Представьте, что `MorseTranslator` — это **двуязычный словарь**. Это как книга, в которой первая половина переводит с английского на Морзе, а вторая — с Морзе на английский. Когда приложению нужно узнать, как "звучит" слово "HELLO", оно открывает эту "книгу".
