# EliMorse — приложение для изучения и перевода азбуки Морзе

## 📘 Обзор

**EliMorse** — многофункциональное Android-приложение, созданное для изучения, практики и использования азбуки Морзе.  
Оно сочетает обучающие тренажёры, инструменты перевода и полезные справочные материалы.

---

## ✨ Основные возможности

### 🔤 Перевод
- **Текст → Морзе** — преобразует английский текст в код Морзе.  
- **Морзе → Текст** — переводит введённый код в обычный текст.

### 📡 Передача сигналов
- **Аудио-воспроизведение** — проигрывает точки и тире со стандартной длительностью.  
- **Передача фонариком** — визуальная передача кода Морзе через вспышку устройства.

### 🧠 Тренировка и обучение
- **Практика кодирования** — упражнения по переводу букв, цифр и слов в Морзе.  
- **Режим “На слух”** — приложение воспроизводит сигнал, пользователь вводит ответ.  
- **Статистика ошибок** — фиксирует неверные ответы для анализа прогресса.

### 👤 Пользовательские функции
- **Регистрация и вход** — встроенная система аутентификации.  
- **История упражнений** — сохранение результата каждого прохождения.

### 🎨 Интерфейс и удобство
- **Светлая / тёмная / системная темы**  
- **Виброотклик** при взаимодействиях  
- **“Умные” кнопки** — автоматически отключаются, если функция устройства недоступна  
- **Оптимизация ресурсов** — корректная работа с `TextToSpeech`, `BroadcastReceiver` и другими компонентами без утечек памяти

### 📚 Справочные материалы
- Полная **таблица кодов Морзе**  
- Раздел **«О приложении»** — контакты разработчика

## 🛠 Архитектура и технологии

- **Язык:** Java  
- **Платформа:** Android  
- **UI:** XML + Material Design  
- **База данных:** SQLite  
- **Компоненты:** Activities, Services, BroadcastReceivers, собственные UI-элементы  
- **Структурирование проекта:**  
  - `MorsePlayer` — проигрывание последовательностей Морзе  
  - `QuestionRepository` — генерация вопросов для обучения  
  - `HelperDB` — управление базой данных
