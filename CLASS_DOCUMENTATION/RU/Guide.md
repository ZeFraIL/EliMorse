# Класс: Guide

## 1. Общая информация
*   **Имя класса:** `Guide`
*   **Тип:** Activity
*   **Назначение:** Этот экран служит дополнительным меню для обучающего раздела приложения. Он предоставляет кнопки для перехода к различным образовательным ресурсам: «Что такое азбука Морзе?», «Как ею пользоваться?» (видео) и «Словарь символов Морзе».
*   **Взаимодействие:** Наследует `BaseActivity`. Открывает экраны `GuideWhat`, `GuideHow` или `GuideList`.

## 2. Переменные (поля класса)
| Имя | Тип | Цель | Где используется |
| :--- | :--- | :--- | :--- |
| `user` (наследуется) | `User` | Данные текущего пользователя. | Передаются в поддиректории справочника. |

## 3. Методы класса

### Имя метода: `onCreate`
*   **Тип метода:** `protected`
*   **Логика работы:**
    1. Устанавливает макет `activity_guide.xml`.
    2. Инициализирует переменные через `initElements()`.
    3. Настраивает обработчики кликов для трех кнопок: `bWhat`, `bHow` и `bList`. Каждая кнопка запускает соответствующее Activity.
    4. Настраивает всплывающее меню для текста `tvMenuGuide`.
*   **Когда вызывается:** Когда пользователь нажимает «Справочник» (Guide) на главной панели.

### Имя метода: `initElements`
*   **Тип метода:** `private`
*   **Логика работы:** Извлекает объект `User` из переданных данных (Intent).
*   **Когда вызывается:** Во время работы `onCreate`.

## 4. Жизненный цикл
*   **`onCreate()`**: Подготавливает экран навигации по справочнику.

## 5. Взаимодействие с интерфейсом (UI)
*   **Кнопки:** `bWhat`, `bHow`, `bList`. Используются для перехода между разделами.
*   **TextView (`tvMenuGuide`):** Служит кнопкой вызова меню.

## 6. Взаимодействие с другими компонентами
*   **Intents:** Используются для запуска специализированных экранов справочника.

## 7. Общая логика класса
Класс `Guide` — это **«Указатель»**. Он сам не содержит учебной информации, а лишь направляет пользователя к тому типу контента, который ему нужен (чтение, просмотр видео или поиск конкретных символов).

## 8. Упрощенное объяснение
Представьте `Guide` как **холл библиотеки**. Когда вы заходите в библиотеку, вы видите указатели: «Читальный зал» (GuideWhat), «Медиатека» (GuideHow) и «Справочное бюро» (GuideList). Вы нажимаете на кнопку, чтобы пройти в нужный зал.
