# 📱 Документация Translate (Активность перевода)

## 1. Общая информация
*   **Название:** `Translate`
*   **Тип:** Активность (Наследует `BaseActivity`)
*   **Назначение:** Этот экран позволяет переводить текст в азбуку Морзе и наоборот. Также он управляет оборудованием (фонариком и звуком) для передачи сигналов.
*   **Взаимодействие:** Использует `MorseTranslator` для логики и `TTS_Service` для озвучки.

## 2. Переменные (Поля класса)
| Название | Тип | Назначение | Использование |
| :--- | :--- | :--- | :--- |
| `translator` | `MorseTranslator` | "Мозг" переводчика. | Выполняет само преобразование текста. |
| `soundPool` | `SoundPool` | Аудио-плеер. | Играет звуки "точки" и "тире" мгновенно. |
| `what_switch` | `int` | Счетчик режима. | Четное = Текст в Морзе; Нечетное = Морзе в Текст. |
| `handler` | `Handler` | Менеджер времени. | Управляет задержками между вспышками и звуками. |

## 3. Методы
### Метод: `playSequence`
*   **Тип:** `private`
*   **Параметры:** `morseCode` (строка), `index` (номер символа).
*   **Что делает:** Это **рекурсивный** метод. Он читает код Морзе по одному символу. Если видит `.`, играет точку; если `-`, играет тире. После этого он вызывает сам себя для следующего символа через нужную паузу.

### Метод: `flash`
*   **Что делает:** Включает вспышку камеры на определенное время (длительность точки или тире) и выключает её.

## 8. Объяснение простыми словами
Представьте, что этот класс — **Телеграфист**. Он берет ваше письмо, смотрит код в справочнике (`MorseTranslator`), а затем отстукивает его ключом (звук) или мигает сигнальным фонарем.
