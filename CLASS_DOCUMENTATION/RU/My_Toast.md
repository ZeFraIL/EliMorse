# Класс: My_Toast

## 1. Общая информация
*   **Имя класса:** `My_Toast`
*   **Тип:** Обычный класс (Утилита)
*   **Назначение:** Этот класс используется для показа стилизованных всплывающих сообщений («Toast»). Вместо стандартного серого текста Android, он создает уведомления с иконкой, особым шрифтом и выравниванием по центру.
*   **Взаимодействие:** Может вызываться из любой Activity в приложении.

## 2. Переменные (поля класса)
*   **Отсутствуют:** Все методы класса являются статическими.

## 3. Методы класса

### Имя метода: `showToast`
*   **Тип метода:** `public static`
*   **Возвращаемое значение:** `void`
*   **Параметры:**
    | Имя | Тип | Описание |
    | :--- | :--- | :--- |
    | `context` | `Context` | Контекст, в котором показывается сообщение. |
    | `message` | `String` | Текст сообщения. |
    | `iconResId`| `int` | (Опционально) ID ресурса иконки. |
*   **Логика работы:**
    1. Использует `LayoutInflater` для загрузки кастомного макета `my_toast.xml`.
    2. Находит элементы `ImageView` (иконка) и `TextView` (текст) в этом макете.
    3. Устанавливает текст сообщения и иконку (если она передана).
    4. Создает новый объект `Toast`, устанавливает его длительность, центрирует на экране и показывает.
*   **Когда вызывается:** Когда приложению нужно дать пользователю быструю обратную связь (например, «Пользователь не найден» или «Низкий заряд батареи»).

## 4. Жизненный цикл
*   **Неприменимо:** Обычный статический помощник.

## 5. Взаимодействие с интерфейсом (UI)
*   **Макет (`my_toast.xml`):** Определяет закругленный фон, положение иконки и стиль шрифта для сообщений.

## 6. Взаимодействие с другими компонентами
*   **LayoutInflater:** Используется для превращения дизайна из XML-файла в живой объект Java во время работы программы.

## 7. Общая логика класса
`My_Toast` — это **«Изготовитель вывесок»**. Вместо того чтобы использовать стандартные таблички, предоставляемые системой Android, приложение использует этот класс для создания уведомлений, которые соответствуют стилю Eli Morse.

## 8. Упрощенное объяснение
Представьте `My_Toast` как **красивую наклейку**. Обычные уведомления Android похожи на простые белые стикеры. `My_Toast` позволяет вам выбрать цветной стикер с картинкой (иконкой), чтобы сообщение выглядело более дружелюбно.
