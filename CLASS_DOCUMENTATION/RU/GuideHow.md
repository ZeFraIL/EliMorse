# Класс: GuideHow

## 1. Общая информация
*   **Имя класса:** `GuideHow`
*   **Тип:** Activity
*   **Назначение:** Этот экран предоставляет визуальное руководство по использованию азбуки Морзе или самого приложения. Он встраивает видео с YouTube с помощью компонента `WebView`.
*   **Взаимодействие:** Наследует `BaseActivity`.

## 2. Переменные (поля класса)
| Имя | Тип | Цель | Где используется |
| :--- | :--- | :--- | :--- |
| `videoID` | `String` | Уникальный идентификатор видео на YouTube. | Используется для создания HTML-кода вставки. |
| `webView` | `WebView` | Компонент для отображения веб-контента. | Используется для загрузки видеоплеера. |

## 3. Методы класса

### Имя метода: `onCreate`
*   **Тип метода:** `protected`
*   **Логика работы:**
    1. Устанавливает макет `activity_guide_how.xml`.
    2. Инициализирует переменные.
    3. Настраивает кнопку «Назад» (`bBack`) для закрытия экрана.
    4. Получает `videoID` из ресурсов строк.
    5. Настраивает `WebView`: включает поддержку JavaScript (необходим для работы YouTube).
    6. Создает строку HTML, содержащую тег `<iframe>` с URL-адресом видео.
    7. Загружает этот HTML-код в `WebView`.
*   **Когда вызывается:** Когда пользователь нажимает «Как использовать» в меню справочника.

## 4. Жизненный цикл
*   **`onCreate()`**: Загружает видеоплеер при создании экрана.
*   **`onPause()` / `onDestroy()`**: Важно останавливать работу WebView при выходе из экрана (примечание: это отсутствует в текущем коде).

## 5. Взаимодействие с интерфейсом (UI)
*   **WebView (`webView`):** Служит контейнером для плеера YouTube.
*   **Button (`bBack`):** Закрывает текущий экран.

## 6. Взаимодействие с другими компонентами
*   **Интернет:** Для работы `WebView` и загрузки видео требуется активное интернет-соединение.

## 7. Общая логика класса
Класс `GuideHow` — это **«Видео-плеер»**. Он использует крошечный «веб-браузер» внутри приложения для показа конкретного видео из интернета. Он упаковывает видео в HTML-код, чтобы оно идеально вписалось в экран мобильного телефона.

## 8. Упрощенное объяснение
Представьте `GuideHow` как **мини-телевизор**. Вместо того чтобы читать об азбуке Морзе, вы можете посмотреть видео о ней. Приложение открывает небольшое окно, которое подключается к YouTube и проигрывает для вас урок.

---
**Советы по улучшению:**
*   **Видео в фоне:** В текущем коде, если пользователь свернет приложение, видео может продолжить играть в фоновом режиме. Лучше вызывать `webView.onPause()` в методе `onPause()` активности.
*   **Проверка интернета:** Если интернета нет, WebView покажет ошибку. Добавление проверки соединения улучшит работу приложения.
